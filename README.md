### 📑 Project Summaries

#### 1. Worldwide Importers Dashboard
An interactive two-page Power BI dashboard designed to analyze over 26,000 sales transactions. Features a high-performance Star Schema data model, comprehensive ETL processes via Power Query, advanced DAX measures, dynamic SWITCH logic for regional performance classification, and dynamic business forecasting using What-If analysis and a live currency exchange API.

#### 2. Water Potability Classifier
An end-to-end machine learning model engineered to predict drinking water safety based on physicochemical features. Features rigorous Exploratory Data Analysis (EDA) and outlier elimination, class imbalance resolution via SMOTE, hyperparameter optimization via GridSearchCV with 5-fold cross-validation (achieving 71.13% test accuracy), and seamless deployment as an interactive Streamlit web application.
